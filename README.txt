Restaurant Menu & Order Tracker
--------------------------------
Database: MongoDB Atlas (NoSQL)
Frontend: HTML + JavaScript
Connection: MongoDB Atlas Data API

Features:
1. Add Menu Items
2. Add Customers
3. Place Orders
4. Calculate Total Bill
5. View All Orders

Collections:
- MenuItems (name, category, price)
- Customers (name, phone, email)
- Orders (customerId, items, totalAmount, status, orderDate)

Demonstrated Concepts:
- CRUD operations
- Relationships using ObjectId
- Data Aggregation (Total Bill)
- Database Connectivity using Data API

How to Run:
1. Open index.html in any browser
2. Make sure MongoDB Data API is enabled and keys are correctly set
3. Perform operations directly from the webpage
